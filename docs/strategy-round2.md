# Round 2 Strategy Merge

The second document pack expands LuxVeritas.media from a polished public front door into an operating system for releases, rights, campaign management, and command-center governance.

## New Strategic Inputs

- `Lux_Veritas_Release_Operations_Playbook-1.docx`: release timelines, locked splits, DSP readiness, landing page QA, asset packs, paid media, and week-one ops.
- `Lux_Veritas_Rights_Royalties_GHL_Build_Sheet-2.docx`: two-sides music model, composition/master governance, PRO/MLC/SoundExchange/Content ID, quarterly audit, fan engine, and exact GHL naming conventions.
- `Lux_Veritas_Campaign_Manager_SOP_v1_1-1.docx`: ad-safe campaign wrapper logic, SPMVP hub rules, UTM schema, platform playbooks, consent, reporting, and budget pacing.
- `LUX_OS_v2_ELITE_Command_Center.xlsx`: leadership dashboard, project portfolio, content pipeline, capture log, GHL funnel board, KPI scorecard, migration tracker, and operating cadence.
- `Lux_Veritas_Codex_Prompt_Pack.md`: production target architecture for Next.js, Firebase, Supabase, GoHighLevel, Stripe, Printify, portal routes, Codex tier gates, and deployment hardening.

## Site Changes Made

- Converted the single-page prototype into a multi-page static site with 33 generated routes.
- Preserved six-item public nav: Home, Music, Film, Events, Codex, About.
- Moved Portal and utilities to footer/route-ready architecture.
- Added event microsite shells.
- Added Codex Inner/Sanctum gate shells.
- Added portal, library, releases, admin, and tier-manager shells.
- Added SPMVP ad-safe campaign hub.
- Added route-ready Works, Brands, Store, Insights, Membership, Investor, and Community pages.
- Added consent banner, UTM persistence, local submission ledger, and analytics event stubs.
- Added Firebase/static hosting config, future App Hosting config, `.env.example`, project instructions, robots, and sitemap.
